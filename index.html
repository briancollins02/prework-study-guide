HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element is contained within the head element.
  • The head and meta elements are not directly viewed by the user.
  • This list is an example of an unordered list <ul>.
  • Items within an unordered list are listed using <li>.
  • to create a line-break, write <br>
  • I will continue to add itmes as I learn!

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSS box model lets you add styles based on tags and attributes, called in the HTML and referenced in CSS.
  • A CSS style sheet must be linked to the HTML in the 'head' code.
  • to apply a style to multiple selectors, list them both with a comma inbetween,
    ie: header, footer {color:blue}. this would change the text color of both header and footer to blue.

Git

  • Every git command begins with the tag "git".
  • git status: checks what branch you are currently working on.
  • git checkout -b branch-name: creates a new branch with given name and switches to it.
  • git checkout branch-name: moves to the named branch.
  • git pull: pulls most recent version from the cloud to your local machine.
  • git add-A: commits all files in the working branch.
  • git commit -m "insert description of changes": adds a message to the commit.
  • git pull origin main: pulls base branch into the feature branch to sync them.
  • git push: pushes code from local machine to the cloud. requires a push request to commit.
